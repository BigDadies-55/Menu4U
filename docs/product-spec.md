# Menu4U — מסמך אפיון מוצר

**גרסה:** 1.0  
**תאריך:** יוני 2026  
**סטטוס:** מערכת פעילה בפיתוח

---

## מה זה Menu4U?

Menu4U היא פלטפורמת ניהול מסעדות דיגיטלית מקצה לקצה — מתפריט דיגיטלי ועד ניהול הזמנות, משמרות, מטבח, נוכחות, נאמנות לקוחות, ו-CRM — הכל ממסך ניהול אחד מרוכז.

המערכת מיועדת למסעדות בודדות, רשתות, ומקומות אירוח — עם תמיכה מלאה בעברית ו-RTL.

---

## 1. ניהול תפריט דיגיטלי

### מה קיים
- **ריבוי תפריטים** למסעדה — כל תפריט עם לוח זמנים נפרד (ימים + שעות)
- **קטגוריות ופריטים** עם תמונות, תיאורים, מחירים, אלרגנים, תגיות תזונה (טבעוני, ללא גלוטן וכו׳)
- **קבוצות מוספים** (Modifiers) — אפשרויות בחירה, חובה/רשות, מגבלות כמות, תוספות מחיר
- **תצוגה ציבורית** דרך QR — עיצוב Elegant או Standard, עם פלטות צבע ועיצוב מותאם אישית
- **תזמון אוטומטי** — תפריט ייום מוצג בשעות הארוחה, תפריט ערב בשעות הערב
- **רב-לשוניות** — תמיכה בתרגומים לכל פריט ותפריט
- **מעקב צפיות** — כמה פעמים הלקוח פתח את התפריט

### מה ייחודי
- תצוגה ציבורית מלאה — ללא הורדת אפליקציה, דרך QR בלבד
- כל שינוי עולה באופן מיידי ללקוחות ללא deploy
- לוח זמנים גמיש — תפריטים שונים לשישי/שבת, ארוחת בוקר/צהריים/ערב אוטומטי

---

## 2. ניהול הזמנות

### מה קיים
- **מחזור חיים מלא:** PENDING → CONFIRMED → PREPARING → READY → DELIVERED / PAID / CANCELLED
- **ניהול שולחנות** — מספר שולחן, מספר סועדים, שם לקוח, טלפון, הערות
- **ניהול קורסים** — אפשרות להחזיק פריטים ("Hold") ולשגר ("Fire") מנה עם מנות
- **ביטול ופיצוי** — Void (ביטול לאחר הגשה) ו-Comp (פיצוי ללקוח) עם ניהול הרשאות ו-PIN מנהל
- **מעקב מקור הזמנה** — לקוח עצמאי, מלצר, טלפון
- **לוג סטטוס** — כל שינוי מתועד עם זמן וזהות
- **מוניה הזמנות** — מספרי הזמנה עוקבים לכל מסעדה

### מה ייחודי
- לקוח יכול להזמין ישירות דרך תפריט QR — ללא אינטראקציה עם מלצר
- עדכוני סטטוס בזמן אמת דרך SSE
- הזמנות מקושרות לחבר מועדון אוטומטית עם צבירת נקודות

---

## 3. מערכת KDS — מסך מטבח

### מה קיים
- **4 ממשקי KDS שונים** לבחירה:
  - **Kanban** — לוח עמודות ויזואלי לפי סטטוס
  - **Ticket Board** — כרטיסיות קלאסיות של מסעדה
  - **Table View** — מבט לפי שולחן
  - **Standard** — תצוגת רשימה בסיסית
- עדכונים בזמן אמת (SSE)
- סימון פריטים מוכנים, ניהול קורסים
- הצגת אלרגנים
- Auto-Ready לקטגוריה — כשמסמנים קטגוריה מוכנה, כל פריטיה מוכנים אוטומטית

### מה ייחודי
- 4 ממשקים שונים — מתאים לכל סגנון מטבח
- תצוגת DISPLAY ייעודית — תפקיד נפרד שמכוון ישירות למסך המטבח בכניסה

---

## 4. POS מלצר חכם

### מה קיים
- ממשק מלצר מותאם לטאבלט/מובייל
- בחירת שולחן ממפת שולחנות ויזואלית
- גלישה בתפריט עם תמונות
- הוספת פריטים עם מוספים
- הגשה לתשלום
- Comp ו-Void עם אישור PIN מנהל
- **תחנות מלצר** — כל מלצר רואה רק את שולחנותיו
- **שעון נוכחות** — כניסה ויציאה עם הערה אופציונלית
- Widget של פריטים פופולריים

### מה ייחודי
- ממשק בהתאמה מלאה לעברית RTL
- תמונת רקע מותאמת למסעדה לסביבת עבודה נעימה
- שעון נוכחות משולב — הכל בכפתור אחד

---

## 5. ניהול שולחנות ומפה

### מה קיים
- **בניית פריסת שולחנות** — גרור-ושחרר, עיצוב ויזואלי
- **מפה חיה** — תצוגה בזמן אמת של כל שולחן (פנוי / תפוס / ממתין לתשלום)
- **ציר זמן שולחנות** — היסטוריה כרונולוגית של ישיבות
- **מעקב ישיבות** — זמן פתיחה/סגירה, סה״כ הכנסות, מספר הזמנות

---

## 6. ניהול משמרות

### מה קיים
- **לוח משמרות** — תצוגה שבועית/חודשית לכל עובד
- **4 סוגי משמרת** בצבעים — בוקר, צהריים, ערב, לילה (מותאם אישית)
- **שליחת לוח משמרות במייל** — לעובד בודד או לכולם, עם עיצוב תואם צבעי המשמרות
- **בקשות החלפה** — עובד מבקש, מנהל מאשר/דוחה
- **מנהל משמרת** — ממשק ייעודי לתפעול יומי
- **נוכחות** — כניסה/יציאה עם הערה, מוגן מכניסה כפולה

### מה ייחודי
- לוח ויזואלי בצבעים עם תמיכה מלאה ב-RTL
- שליחת מייל HTML מעוצב עם צבעי המשמרות — לא PDF משעמם

## 7. מועדון לקוחות ונאמנות

### מה קיים
- **הרשמה** — לקוח נרשם דרך התפריט הציבורי עם טלפון
- **צבירת נקודות** — לפי יחס שקל/נקודה מוגדר
- **מימוש נקודות** — המרה חזרה לשקלים בהזמנה
- **קופונים** — % הנחה, סכום קבוע, פריט חינם; עם תוקף ומעקב שימוש
- **בונוס הצטרפות + יום הולדת**
- **רמות פעילות** — New / Active / At Risk / Inactive
- **ניהול מרוכז לרשת** — נקודות ניתן לממש בכל סניף בקבוצה
- **הרשאות גמישות** — מי יכול לתת נקודות ידנית, מי יכול להנפיק קופון, מי ממש

### מה ייחודי
- מועדון נוכח בממשק ציבורי — לקוח מצבר נקודות בלי להוריד אפליקציה
- שיתוף נקודות בין סניפי הרשת
- כל פעולה מבוקרת עם יומן ביקורת

## 8. CRM ותקשורת לקוחות

### מה קיים
- **שליחת SMS** — Bulk ל-כלל חברי המועדון דרך ספק Inforu
- **תבניות הודעה** עם שדות אישיים ([#Name#] וכו׳)
- **היסטוריית SMS** — מעקב כל שליחה
- **קמפיינים** — יצירה, תזמון, ביצוע
- **מיקוד לקהלים** — פילטור לפי סטטוס חבר, תאריך, הוצאות
- **מייל** — איפוס סיסמה, הזמנה למערכת, קבלת משמרות, ברוך הבא

## 9. ניהול משתמשים והרשאות

### מה קיים
- **8 תפקידים:**

| תפקיד | מה הוא עושה |
|--------|-------------|
| SUPER_ADMIN | גישה מלאה לכל המערכת |
| ADMIN | ניהול מלא פרט ל-Super Admin |
| OWNER | ניהול מסעדתו בלבד |
| SHIFT_MANAGER | תפעול, משמרות, מטבח, הזמנות |
| EDITOR | עריכת תפריטים בלבד |
| WAITER | POS מלצר + נוכחות |
| VIEWER | צפייה בלבד |
| DISPLAY | מסך מטבח בלבד |

- **שיוך למסעדות** — אותו משתמש יכול להיות OWNER בסניף אחד ו-EDITOR בשני
- **מטריצת הרשאות** — מסך ויזואלי שמציג מי רשאי לגשת למה
- **אבטחת חשבון** — נעילה אחרי 5 ניסיונות, 2FA למנהלים, PIN מנהל לפעולות רגישות
- **מדיניות סיסמאות** — אורך מינימלי, היסטוריה, גיל מקסימלי, דרישות תווים
- **יומן ביקורת** — כל פעולה מתועדת עם משתמש, IP, זמן

## למה לעבוד עם Menu4U?

### 1. הכל במקום אחד
מרוב המערכות בשוק צריך לחבר כלים שונים — מערכת הזמנות נפרדת, KDS נפרד, מועדון נאמנות נפרד, לוח משמרות נפרד. Menu4U כוללת את הכל משולב.

### 2. מותאמת לישראל
- עברית מלאה + RTL בכל ממשק
- שקלים, אזור זמן ישראל, פורמט תאריכים ישראלי
- SMS דרך ספקים ישראליים (Inforu)
- תמיכה בחגי ישראל בשעות פתיחה

### 3. ללא אפליקציה ללקוח
לקוח מקבל תפריט, מזמין, מצבר נקודות — הכל דרך QR בדפדפן. אין הורדה, אין הרשמה מסורבלת.

### 4. גמישות לרשתות
מספר סניפים, קבוצות מסעדות, ניהול מרוכז עם בידוד נתונים בין סניפים — תמיכה מובנית.

### 5. אבטחה ברמה ארגונית
2FA, נעילת חשבון, PIN מנהל, יומן ביקורת מלא, מדיניות סיסמאות — אבטחה שמסעדות גדולות דורשות.

### 6. זמן אמת
הזמנות, KDS, מפת שולחנות — הכל מתעדכן אוטומטית ללא רענון דף.

### 7. Open & Scalable
המערכת בנויה על Next.js, PostgreSQL, Prisma — טכנולוגיות סטנדרט תעשייתי. ניתן לפתח, לשנות, ולהרחיב.

## סיכום טכני

| פרמטר | פרטים |
|--------|--------|
| Framework | Next.js 15 (App Router) |
| שפת תכנות | TypeScript |
| בסיס נתונים | PostgreSQL + Prisma ORM |
| Authentication | NextAuth.js |
| Email | Gmail SMTP (Nodemailer) |
| SMS | Inforu API v2 |
| עדכונים בזמן אמת | Server-Sent Events (SSE) |
| Push Notifications | Web Push API (VAPID) |
| עיצוב | Dark Glass Morphism, RTL מלא |
| Deployment | Node.js / Docker / Vercel |
