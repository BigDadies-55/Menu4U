# כיצד לשנות את פלטת הצבעים של האתר

## הקובץ הבודד לשינוי

```
src/lib/ui.ts
```

זה **המקום היחיד** שצריך לערוך. כל הדפים, כל הקומפוננטות, וכל התבניות קוראות את הצבעים משם.

---

## מה *לא* נשלט מ-ui.ts

### ערכות נושא מנהל (Admin Themes)

ב-`src/components/admin/Sidebar.tsx` קיים `ADMIN_PALETTE_MAP` עם 5 ערכות נושא שהמשתמש יכול לבחור בממשק (כהה / סגול / כחול / ירוק / ורוד).

ה-`bg` של כל ערכת נושא (כגון `#120b1e` לסגול, `#080f1e` לכחול) **נשאר כצבע גולמי** בכוונה — זה מה שמפריד בין הערכות. ה-`accent` וה-`accentText` שלהן כן מגיעים מ-T.

אם רוצים לשנות גם את הצבעים שם:

```ts
// src/components/admin/Sidebar.tsx → ADMIN_PALETTE_MAP
export const ADMIN_PALETTE_MAP = {
  dark:   { bg: "#...", accent: T.gold, ... },
  purple: { bg: "#...", accent: T.purple, ... },
  // ...
}
```

## תהליך השינוי המומלץ

1. פתח `src/lib/ui.ts`
2. שנה את ערכי ה-`bg`, `surface`, `panel`, `text`, `gold` (אלה הכי משפיעים)
3. שמור את הקובץ
4. הדפדפן מתעדכן אוטומטית (hot-reload)
5. בדוק את הדפים: `/admin`, `/preview/dashboard`, `/preview/waiter`, `/preview/shift`

## טוקנים לפי חשיבות

| טוקן | משמעות | דוגמה לשינוי |
|------|---------|--------------|
| `bg` | רקע כל הדפים | הצבע הכהה ביותר |
| `surface` | כרטיסים, פאנלים | קצת בהיר יותר מ-bg |
| `gold` | הצבע הראשי / מותג | כפתורים, הדגשות, active states |
| `text` | כל הטקסט הראשי | צבע בהיר (על רקע כהה) |
| `sub` | תוויות ומשנה | בינוני |
| `muted` | placeholder, מושבת | הכהה מבין הטקסטים |
| `green` | פנוי, הצלחה | ירוק |
| `red` | שגיאה, חשבון | אדום |
| `orange` | תפוס, אזהרה | כתום |
| `bgNeon` | רקע תבניות נאון | כחול-שחור עמוק |
